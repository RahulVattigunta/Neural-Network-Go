# 🚀 Go Neural Network

This project is a simple neural network implemented in Go, built from scratch using the `gonum` library. The neural network is trained on the Iris dataset to classify flowers into three species.

## 🏗 Tech Stack
- **Language:** Go 🐹
- **Libraries:** Gonum for matrix operations 📊
- **Development Environment:** VS Code / GoLand / Terminal 🖥
- **Version Control:** Git & GitHub 🛠

## 📌 Features
- 📦 **Neural Network:** Implements a simple feedforward neural network.
- 🔢 **Matrix Operations:** Uses `gonum` to handle mathematical computations.
- 🔗 **Backpropagation:** Trains the model using gradient descent.
- 📜 **Iris Dataset:** Uses a modified Iris dataset for classification.

## 🛠 Setup & Run

### 1️⃣ Install Go
Ensure you have Go installed on your system. You can verify with:
```sh
go version
```

### 2️⃣ Clone This Repository
```sh
git clone https://github.com/your-username/Go_NeuralNet.git
cd Go_NeuralNet/src
```

### 3️⃣ Install Dependencies
```sh
go mod init Go_NeuralNet
go get gonum.org/v1/gonum/mat
```

### 4️⃣ Run the Program
```sh
go run main.go
```
